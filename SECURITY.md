# Security Policy

## Scope

Zoryu Trade Core contains **domain logic only**.
It does not handle private keys, custody, transaction signing, or on-chain execution.

## Reporting

If you discover a security issue affecting:
- validation bypass
- incorrect lifecycle evaluation
- inconsistent state transitions
- incorrect PnL/metrics computation

Please report it privately to the maintainers.

## Notes

Do not open public issues for vulnerabilities that could be abused.
